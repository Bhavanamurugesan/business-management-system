# business-management-system
Full Stack Business Management System using Flask and SQLite with Customer, Product, Inventory, Sales, and User Management modules.A simple Business Management Web Application built using Python Flask, SQLite, HTML, CSS, and JavaScript.

## Features

- Customer Management
  - Add Customers
  - Edit Customers
  - View Customer Details

- Product Management
  - Add Products
  - Edit Products
  - Manage Product Details

- Sales Management
  - Record Sales
  - Track Transactions

- User Management
  - Add Users
  - Manage User Information

- Database Integration using SQLite

- Responsive Frontend using HTML, CSS, and JavaScript

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/BusinessManagementApp.git
```

### 2. Open Project Folder

```bash
cd BusinessManagementApp
```

### 3. Install Flask

```bash
pip install flask
```

### 4. Run Application

```bash
python app.py
```
